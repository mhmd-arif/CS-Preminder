# Preminder
Preminder adalah sebuah aplikasi yang dapat membuat daftar kegiatan apa saja yang harus kita kerjakan. Aplikasi ini diharapkan dapat membantu penggunanya untuk dapat mengatur jadwal dari pengguna.
